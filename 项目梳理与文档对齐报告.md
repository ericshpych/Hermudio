# Hermudio 项目梳理与文档对齐报告

## 一、项目概述

**Hermudio** 是一个 AI 个性化音乐电台项目，通过 AI DJ 自动播报开场白、歌曲介绍、过渡文案，支持语音播报，提供电台模式和聊天模式两种交互方式。

---

## 二、项目结构

```
Hermudio/
├── Hermudio/                          # 主项目目录
│   ├── server.js                      # Express 服务器（端口6688）
│   ├── package.json                   # 依赖配置
│   ├── src/
│   │   ├── music-service.js           # 音乐服务（ncm-cli集成）
│   │   ├── recommendation-engine.js    # 推荐引擎
│   │   ├── scene-analyzer.js          # 场景分析器
│   │   ├── hermes-service.js          # Hermes AI 服务集成
│   │   ├── radio-host-service.js       # 电台主持服务
│   │   ├── radio-host.js              # 电台主持（旧版）
│   │   ├── tts-service.js             # TTS 服务
│   │   └── user-profile.js            # 用户画像服务
│   ├── public/
│   │   ├── index.html                 # 主页面（电台+聊天双模式）
│   │   ├── sw.js                      # Service Worker
│   │   └── mock-test.html             # 测试页面
│   └── data/
│       └── hermudio.db                # SQLite 数据库
├── .ncm-home/                         # ncm-cli 配置目录
├── .trae/documents/                   # 项目文档
└── qqmusic_playlists/                 # QQ音乐歌单数据
```
