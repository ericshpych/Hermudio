# Hermudio TTS 优化方案

## 📋 当前问题

1. **MiniMax TTS 依赖**：流量有限，不适合打包使用
2. **硬编码 API Key**：安全风险
3. **没有默认免费方案**：未充分利用 Web Speech API

## 🚀 优化方案

### 方案一：优先使用 Web Speech API（推荐）

**优势**：
- ✅ **完全免费**，无流量限制
- ✅ **本地运行**，无需网络请求
- ✅ **隐私安全**，数据不发送到服务器
- ✅ **响应快速**
- ✅ **支持多种语言和音色

**实现**：
- 已存在于 `src/tts-service.js`
- 在前端默认使用 Web Speech API
- 仅在需要高质量时（如有需求）使用 MiniMax 作为备选

### 方案二：TTS 缓存机制

**功能**：
- 缓存常用的播报文本
- 避免重复合成相同的内容
- 节省流量和成本

### 方案三：多 TTS 服务支持

**支持的服务**：
1. **Web Speech API**（默认，免费）
2. **MiniMax**（高质量，付费）
3. **Edge TTS**（免费，使用 Microsoft 服务）
4. **Coqui TTS**（本地模型）
